Liste des commandes git
Commande Descritpion
git status Status des fichiers
git branch -a Lister les branchs
git checkout -b nom_de_ma_branch_nouvelle Créer une branch
git branch -d nom_de_ma_branch_local Supprimer une branch
git checkout nom_de_ma_branch Changer de branch
git commit -m "initial commit" Premier commit
git commit -m "message du commit" Commit suivant
git pull Mettre à jour le dépôt local
git push Envoyer ses commits vers le dépôt distant
